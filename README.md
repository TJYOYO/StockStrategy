# 美股趋势上涨策略脚本

## 功能概述

本脚本针对美股市场，基于趋势上涨策略，提供买和卖股票的时机。主要功能包括：

1. **美股数据获取**：通过 Yahoo Finance API 获取实时美股数据
2. **技术指标计算**：计算移动平均线、MACD、RSI、布林带等关键技术指标
3. **买卖信号生成**：基于多重条件生成买入和卖出信号
4. **策略表现分析**：模拟交易并分析策略表现
5. **可视化展示**：生成价格走势图和技术指标图表

## 安装依赖

```bash
pip install -r requirements.txt
```

## 快速开始

### 基本使用

```bash
# 分析苹果股票 (AAPL)
python stock_strategy.py

# 分析特定股票
python stock_strategy.py --symbol MSFT

# 使用不同时间周期
python stock_strategy.py --symbol GOOGL --period 1y

# 显示图表
python stock_strategy.py --symbol AAPL --plot

# 保存图表
python stock_strategy.py --symbol AAPL --save-plot output.png

# 导出结果
python stock_strategy.py --symbol AAPL --export results.json
```

```bash

python batch_analysis.py

# 自定义策略参数
python custom_strategy.py
```

## 策略原理

### 买入信号条件

1. **上升趋势确认**：价格在20日均线之上，且20日均线在50日均线之上
2. **MACD金叉**：MACD线上穿信号线
3. **RSI超卖回升**：RSI从超卖区域（<30）回升
4. **布林带下轨支撑**：价格触及或接近布林带下轨
5. **成交量放大**：成交量显著高于20日平均成交量

### 卖出信号条件

1. **下降趋势确认**：价格在20日均线之下，且20日均线在50日均线之下
2. **MACD死叉**：MACD线下穿信号线
3. **RSI超买回落**：RSI从超买区域（>70）回落
4. **布林带上轨阻力**：价格触及或接近布林带上轨
5. **成交量萎缩**：成交量显著低于20日平均成交量

### 信号强度计算

每个信号条件都有相应的权重，当满足至少3个条件且总强度超过0.5时，生成交易信号。

## 故障排除

### 常见问题

1. **API速率限制错误**：
   ```
   YFRateLimitError: Too Many Requests. Rate limited. Try after a while.
   ```
   **解决方案**：
   - 脚本已内置**数据回退机制**，当检测到API限制时会自动使用模拟数据进行演示
   - 模拟数据包含完整的股票价格走势和技术指标，可以展示策略的全部功能
   - 对于真实交易，建议等待API限制解除后使用真实数据

2. **数据获取失败**：
   - 检查网络连接
   - 验证股票代码是否正确
   - 尝试不同的时间周期（如使用 `--period 1y` 代替 `--period 6mo`）
   - 脚本会自动回退到模拟数据，确保功能演示不受影响

3. **技术指标计算错误**：
   - 确保数据量足够（至少20个数据点）
   - 检查 `ta` 库是否正确安装

### 数据回退系统

脚本包含一个智能的数据回退系统：

1. **优先尝试真实数据**：使用优化的yfinance请求，包含重试机制和User-Agent轮换
2. **自动回退到模拟数据**：当API限制时，自动生成逼真的模拟数据
3. **完整功能演示**：模拟数据支持所有技术指标计算和信号生成
4. **明确标识**：使用模拟数据时会明确提示用户

### 配置说明

`config.json` 中的关键配置：

```json
"data_quality": {
  "min_data_points": 10,      // 最小数据点要求
  "fallback_period": "1mo",   // 备用时间周期
  "validate_data": true       // 是否验证数据质量
}
```

## 注意事项

1. **数据延迟**：Yahoo Finance 数据有15分钟延迟
2. **API限制**：Yahoo Finance API有速率限制，频繁请求可能导致暂时被封禁
3. **历史回测**：策略基于历史数据，过去表现不代表未来结果
4. **风险管理**：实际交易请结合风险管理，本脚本仅供学习和研究使用
5. **参数优化**：可根据市场情况调整策略参数
6. **数据质量**：确保获取足够的数据点（建议至少20个）以进行准确分析
